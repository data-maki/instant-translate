## 2026-05-19 Multilingual Text Translation Goal

- (proved) Hypothesis: the app was mostly hard-coded around Japanese/English display and DeepL Japanese formality, but Soniox/session storage already carried language codes generically. Results: added Burmese (`my`), preserved multiple Soniox language hints, made the English rewrite and DeepL translation path target-language aware, omitted DeepL formality for Thai/Burmese/Vietnamese, added a left-lane language selector, and covered Thai/Burmese/Vietnamese plus unexpected French text tokens in backend tests. Reflection: the smallest useful path was not a new multilingual architecture; it was preserving language hints, removing Japanese-only assumptions from the rewrite/translation layer, and letting the UI filter/render whatever non-English language codes appear.
